De app component bevat in het groene gedeelte een fomulier waarop 2way input binding & one way binding is toegepast. Op de knop is een event gekoppeld die linkt naar de submit() methode in de component klasse. Ook op het inputveld van de naam is een event gekoppeld.

De value van voornaam is {{voornaam}} en de value van achternaam is {{achternaam}}.
